# Retos del Emprendedor en Colombia

Presentación interactiva sobre los retos y obstáculos del emprendedor en Colombia.

## Fase 5 - Apropiación de la legislación empresarial

Este proyecto presenta una aplicación web interactiva que explora los principales desafíos que enfrentan los emprendedores en Colombia en materia laboral, comercial y tributaria.

## Desarrollo Local

### Prerrequisitos

- Node.js (versión 16 o superior)
- npm o yarn

### Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Previsualizar build de producción
npm run preview
```

## Deploy en Vercel

### Opción 1: Deploy desde GitHub

1. Sube el código a un repositorio de GitHub
2. Visita [vercel.com](https://vercel.com)
3. Importa tu repositorio de GitHub
4. Vercel detectará automáticamente el proyecto Vite
5. Haz clic en "Deploy"

### Opción 2: Deploy con Vercel CLI

```bash
# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel
```
